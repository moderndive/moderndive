# Tactile sampling from a tub of balls

Counting the number of red balls in 33 tactile samples of size n = 50
balls from
<https://github.com/moderndive/moderndive/blob/master/data-raw/sampling_bowl.jpeg>

## Usage

``` r
tactile_prop_red
```

## Format

A data frame of 33 rows representing different groups of students'
samples of size n = 50 and 4 variables

- group:

  Group members

- replicate:

  Replicate number

- red_balls:

  Number of red balls sampled out of 50

- prop_red:

  Proportion red balls out of 50
