# Bootstrap resamples of a sample of 50 pennies

35 bootstrap resamples with replacement of sample of 50 pennies
contained in a 50 cent roll from Florence Bank on Friday February 1,
2019 in downtown Northampton, Massachusetts, USA
<https://goo.gl/maps/AF88fpvVfm12>. The original sample of 50 pennies is
available in
[`pennies_sample()`](https://moderndive.github.io/moderndive/reference/pennies_sample.md)
.

## Usage

``` r
pennies_resamples
```

## Format

A data frame of 1750 rows representing 35 students' bootstrap resamples
of size 50 and 3 variables

- replicate:

  ID variable of replicate/resample number.

- name:

  Name of student

- year:

  Year on resampled penny
