# A sampling bowl of red and white balls

A sampling bowl used as the population in a simulated sampling exercise.
Also known as the urn sampling framework
<https://en.wikipedia.org/wiki/Urn_problem>.

## Usage

``` r
bowl
```

## Format

A data frame 2400 rows representing different balls in the bowl, of
which 900 are red and 1500 are white.

- ball_ID:

  ID variable used to denote all balls. Note this value is not marked on
  the balls themselves

- color:

  color of ball: red or white
