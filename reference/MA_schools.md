# Massachusetts Public High Schools Data

Data on Massachusetts public high schools in 2017

## Usage

``` r
MA_schools
```

## Format

A data frame of 332 rows representing Massachusetts high schools and 4
variables

- school_name:

  High school name.

- average_sat_math:

  Average SAT math score. Note 58 of the original 390 values of this
  variable were missing; these rows were dropped from consideration.

- perc_disadvan:

  Percent of the student body that are considered economically
  disadvantaged.

- size:

  Size of school enrollment; small 13-341 students, medium 342-541
  students, large 542-4264 students.

## Source

The original source of the data are Massachusetts Department of
Education reports <https://profiles.doe.mass.edu/state_report/>, however
the data was downloaded from Kaggle at
<https://www.kaggle.com/ndalziel/massachusetts-public-schools-data>
