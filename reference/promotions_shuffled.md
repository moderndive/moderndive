# One permutation/shuffle of promotions

Shuffled/permuted data from a 1970's study on whether gender influences
hiring recommendations.

## Usage

``` r
promotions_shuffled
```

## Format

A data frame with 48 observations on the following 3 variables.

- id:

  Identification variable used to distinguish rows.

- gender:

  shuffled/permuted (binary) gender: a factor with two levels `male` and
  `female`

- decision:

  a factor with two levels: `promoted` and `not`
