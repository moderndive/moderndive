# Massachusetts 2020 vs. 2019 Traffic Data Comparison

This dataset contains information about changes in speed, volume, and
accidents of traffic between 2020 and 2019 by community and class of
road in Massachusetts.

## Usage

``` r
ma_traffic_2020_vs_2019
```

## Format

A data frame of 264 rows each representing a different community in
Massachusetts.

- community:

  City or Town

- functional_class:

  Class or group the road belongs to

- change_in_speed:

  Average estimated Speed (mph)

- change_in_volume:

  Average traffic

- change_in_accidents:

  Average number of accidents
