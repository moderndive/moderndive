- [`tidy_summary()`](moderndive.github.io/moderndive/reference/tidy_summary.md)
  :

  This function calculates the five-number summary (minimum, first
  quartile, median, third quartile, maximum) for specified numeric
  columns in a data frame and returns the results in a long format. It
  also handles categorical, factor, and logical columns by counting the
  occurrences of each level or value, and includes the results in the
  summary. The `type` column indicates whether the data is numeric,
  character, factor, or logical.
